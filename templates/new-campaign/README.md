# New Campaign Template

Copy this folder to `campaigns/<slug>/` and fill in:

- `CAMPAIGN-PLAN.md` — the strategic doc (audience, angle, phases, content schedule)
- `BUILD-INSTRUCTIONS.md` — concrete steps to produce the assets
- `STATUS.md` — current state (read by the CLAUDE.md "Right Now" header — keep this current!)
- `output/` — gitignored, where rendered assets go
- `reels/` — campaign-specific reel compositions (or use the shared `reels/` project)

See `campaigns/summer-camp-2026/` for a fully-built example.
