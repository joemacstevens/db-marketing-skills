# DB Weekly Highlights

45-second Instagram Reel for **Different Breed Elite Fitness**.

## Specs
- **Dimensions:** 1080×1920 (vertical 9:16)
- **FPS:** 30
- **Duration:** 45s (1350 frames)

## Structure

| Segment | Time | Clip |
|---------|------|------|
| Intro Hook | 0–3s | C4323_intro |
| Boxing (bag work) | 3–8s | C4285_boxing_bag |
| Boxing (combos) | 8–12s | C4306_boxing_combos |
| Circuit Training | 12–18s | IMG_4673_training |
| Circuit (BOSU) | 18–22s | IMG_4670_bosu |
| Men to the Mat (magic circle) | 22–29s | C4318_magic_circle |
| Men to the Mat (mat work) | 29–35s | C4286_mat_work |
| Testimonial | 35–43s | C4324_testimonial |
| End Card | 43–45s | DB logo + Evolve tagline |

## Usage

```bash
# Preview in browser
npm start

# Render to file
npm run render
```

## Adding Music
Drop a track in `public/audio/` and add an `<Audio>` component to `WeeklyHighlights.tsx`.

## Brand Colors
- Background: `#1a1a1a`
- Red accent: `#E31937`
- Font: Oswald (Google Fonts)
