# Skill: Social Content Generator

## Purpose
Generate on-brand social media posts for Different Breed Elite Fitness across Instagram and Facebook.

## Before You Start
Read these brand context files:
- `brand-context/voice-and-tone.md`
- `brand-context/coaches-and-staff.md`
- `brand-context/posting-rules.md`
- `brand-context/content-examples.md`

## Input
Provide any of:
- **Topic/theme** ("motivational Monday", "kids boxing promo", "Pilates spotlight")
- **Photo/video description** ("photo of CJ teaching Pilates", "ring action shot")
- **Event details** ("open house this Saturday 10am-2pm")
- **Specific message** ("we're closed for the holiday")

## Output Format
For EVERY post, generate:

```
### Instagram Caption
[Full caption with DB voice, coach tags where relevant]

[Up to 5 hashtags]

### Facebook Caption (≤255 characters)
[Shorter, punchier version of the same message]

### Notes
- Suggested posting time
- Photo/video direction if needed
- Any tagging reminders
```

## Content Types & Frameworks

### Motivational
- Short, punchy sentences. Coach energy.
- Framework: Statement → Challenge → CTA
- Example: "No excuses. Just work. → Are you ready? → Walk through the door."

### Class Promo
- Lead with what makes it special, not just the schedule
- Include: day, time, coach name + handle, booking link if Pilates
- Framework: Hook → What to expect → Details → CTA

### Coach Spotlight
- Warm but still DB-gritty
- Highlight their expertise and what makes their class different
- Always tag the coach
- Framework: Coach intro → What they bring → Schedule → CTA

### Member Spotlight
- Let the transformation/story speak
- Keep it authentic, not testimonial-ad
- Framework: Before/journey → The DB difference → Inspiration

### Event Announcement
- Urgency. "Don't miss this."
- Clear details: what, when, where
- Framework: Attention grab → Event details → Why it matters → CTA

### Behind the Scenes
- Minimal caption. Let the visual do the work.
- Raw, authentic energy
- ALWAYS credit videographer if using their footage
