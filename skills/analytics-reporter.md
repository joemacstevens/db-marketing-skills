# Skill: Analytics Reporter

## Purpose
Track social media performance, generate reports, and recommend strategy adjustments for Different Breed.

## Before You Start
Read these brand context files:
- `brand-context/posting-rules.md`
- `brand-context/target-audiences.md`

## Key Metrics to Track

### Engagement
- **Engagement rate** (likes + comments + shares + saves ÷ reach)
- **Save rate** (saves ÷ reach — strongest signal of valuable content)
- **Comment quality** (real engagement vs. emoji-only)
- **Share rate** (how often content gets shared/reposted)

### Growth
- **Follower growth** (net new per week/month)
- **Reach** (unique accounts that saw content)
- **Impressions** (total views including repeats)
- **Profile visits** (from content → profile)
- **Website clicks** (profile → site/booking)

### Content Performance
- **Top posts** by engagement rate
- **Best time to post** based on engagement data
- **Content type performance** (Reels vs. Carousels vs. Single image vs. Stories)
- **Hashtag performance** (which 5 drive the most reach)

### Platform-Specific
- **Instagram:** Reel plays, Story completion rate, carousel swipe-through rate
- **Facebook:** Post reach, page engagement, new page likes

## Report Formats

### Weekly Quick Check
```
## DB Social — Week of [Date]

### Highlights
- Top post: [description] — [engagement rate]
- Follower change: +/- [number]
- Best performing content type: [type]

### Action Items
- [What to do more of]
- [What to stop/change]
```

### Monthly Recap (for Don)
```
## DB Social Media Report — [Month Year]

### Executive Summary
[2-3 sentences: how did we do, what worked, what's next]

### Key Numbers
| Metric | This Month | Last Month | Change |
|--------|-----------|------------|--------|
| Followers | X | X | +/-X |
| Avg Engagement Rate | X% | X% | +/-X% |
| Total Reach | X | X | +/-X% |
| Website Clicks | X | X | +/-X% |

### Top 5 Posts
1. [Post] — [why it worked]
2. ...

### Content Type Breakdown
[Which types performed best/worst]

### Recommendations
1. [Strategy adjustment]
2. [Content suggestion]
3. [Opportunity identified]

### Next Month Focus
[What to prioritize]
```

### Quarterly Strategy Review
- Deeper trend analysis
- Audience growth patterns
- Competitive benchmarking (optional)
- Content pillar performance
- Resource allocation recommendations

## Analysis Frameworks

### What's Working?
Look for patterns in top-performing content:
- Time of day
- Content type
- Topic/theme
- Caption length
- Visual style
- Coach featured

### What's Not?
Identify underperforming patterns and suggest pivots.

### Opportunities
- Trending formats DB hasn't tried
- Underserved audience segments
- Cross-promotion possibilities
- User-generated content potential

## Quality Checklist
- [ ] Data-driven, not opinion-based?
- [ ] Actionable recommendations (not just "post more")?
- [ ] Formatted clearly for Don (non-technical)?
- [ ] Includes both wins and areas to improve?
- [ ] Specific next steps with timelines?
