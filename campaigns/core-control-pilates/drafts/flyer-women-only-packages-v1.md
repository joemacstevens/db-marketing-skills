### Layout Design Notes
- **Decorative frame:** Double-rule border in warm taupe wrapping full flyer. Small
  bronze diamond ornaments at each corner — directly matching the seed flyers' gold
  border and corner treatment.
- **Top zone:** Brand eyebrow in tracked Montserrat, followed by serif headline and
  italic subhead. Triple-diamond divider separates headline from photo.
- **Hero zone:** Full-width studio photograph (reformer room with warm LED lighting,
  mats and equipment visible). Bronze frame border, subtle rounded corners. This is
  the visual anchor — occupies ~35–40% of flyer height.
- **Mid zone:** Vibe points in 2x2 diamond-bullet layout. Decorative divider.
  Body copy centered below.
- **Pricing zone:** Elevated card on cream with soft shadow. Two tiers with dotted
  leader between label and price. Bronze accent on dollar amounts. Diamond ornaments
  flanking "Class Packages" header.
- **Schedule:** Compact inline with diamond ornaments flanking label. Day names bold,
  times in regular weight.
- **CTA:** Cocoa (#603020) pill button, ivory text, generously padded.
- **Brand footer:** Contact info centered. Grip socks note in muted italic.

## 5) Print Specifications

| Property          | Value                              |
|-------------------|------------------------------------|
| Trim size         | 5" x 7" (portrait flyer/postcard)  |
| Bleed             | 0.125" all sides                   |
| Safe zone         | 0.25" inset from trim              |
| Resolution        | 300 DPI minimum                    |
| Color space       | CMYK for print / sRGB for digital  |
| File formats      | PDF (print), PNG (digital share)   |
| Paper stock rec.  | 100lb gloss cover or 80lb matte    |

### Export Notes
- **Print PDF:** Flatten all layers. Embed fonts (Cormorant Garamond, Montserrat,
  Lora). Include bleed marks and crop marks. Convert to CMYK before output.
- **Digital PNG (primary):** Export at 1500 x 2100 px (300dpi equiv for 5x7). sRGB.
  No bleed/crop marks.
- **Social share (Instagram 4:5):** Re-export at 1080 x 1350 px. Tighten bottom margins.
- **QR code:** If included, minimum 0.75" square. Bottom-right of footer. Test before export.
- **Photo placeholder:** Current HTML uses a gradient placeholder. Replace with actual
  studio photo (reformer room, warm lighting) before final export.
