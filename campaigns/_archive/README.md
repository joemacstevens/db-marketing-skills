# Campaigns — Archive

Completed or abandoned campaigns live here so the live `campaigns/` listing stays scannable.

## What goes here

- Campaigns that wrapped (post-event, post-camp, post-promo)
- Abandoned starts (the campaign equivalent of a false start)
- One-off historical assets (March Madness PDF, etc.)

## Convention

Move the entire campaign folder into `_archive/<original-slug>/` and leave a one-line `ARCHIVED.md` at the top with: date archived, reason, link to any successor campaign.

```
campaigns/_archive/march-madness-2026/
├── ARCHIVED.md       ← "Archived 2026-04-26: one-off historical PDF, no follow-up planned"
└── March Madness.pdf
```
