# Visual Style Guide — Different Breed Elite Fitness

## Color Palette

### Core Colors
| Name | Hex | Usage |
|------|-----|-------|
| Black (base) | `#0E0E0E` | Backgrounds, primary |
| Charcoal (surface) | `#1C1C1C` | Cards, sections, rhythm on black |
| Red (accent/CTA) | `#C4161C` | Actions, emphasis, key highlights, dividers |
| White | `#FFFFFF` | Text on dark, clean sections |
| Gray (muted) | `#8E8E8E` | Secondary text, labels (never body on white) |

### Special Accents
| Name | Hex | Usage |
|------|-----|-------|
| Gold | From brand kit | **RARE** — awards, championships, "Evolve into greatness" moments only |

### Color Rules
- Red = actions + emphasis ONLY. Keep it rare and meaningful.
- Charcoal creates rhythm on black backgrounds (cards/sections).
- Gray for secondary text/labels only, never for body copy on white.
- Gold limited to "badge" moments (awards, credentials, championship callouts). NOT general UI.

## Typography

### Fonts
- **Headlines:** NORD (Black/Bold) — all caps used sparingly
- **Subheads:** NORD Medium
- **Body:** Inter (clean, modern, highly readable)
- **Small UI/labels:** NORD Medium with tracking

### Type Rules
- Avoid walls of all-caps paragraph text
- Use caps for **headlines and labels** only, body in sentence case
- Tight line lengths, strong hierarchy
- Bold uppercase headline moments are on-brand

## Logo System
- **DB circular mark** (red/black/white) — favicon, social profiles
- **DB wordmark** (red+black, red+white) — headers
- **"What we do is absolutely different"** lockups — section headers (sparingly)
- **"Evolve into greatness"** lockups (black/white + gold variants) — footer, premium moments
- **"Strength. Individuality. Elite performance"** lockups

## Photography Style
- **Documentary, high-contrast, "real gym" energy**
- Real sessions: coaches teaching, ring action, turf work, TRX/Pilates studio shots
- Limit stock photos — the DB archive is stronger
- Two acceptable treatments:
  1. **True color** with controlled contrast (skin tones accurate)
  2. **B&W** for hero moments and coach portraits
- **Composition:**
  - Hero images: wide, negative space for headline
  - "Proof" images: close-up faces + emotion
  - Studio images: one establishing + one detail
